EIV Music — שלב 2: איחוד מלא
3 תיקיות = 3 אתרים. גוררים תיקייה שלמה לכל אתר.
════════════════════════════════════════════════════

לפני הכל — אם עוד לא הרצת את ה-SQL משלב 1:
Supabase → SQL Editor → הדבק את supabase-upgrade.sql → Run
(אם כבר הרצת — אפשר להריץ שוב, זה בטוח ולא משנה כלום)

──────────────────────────────────────
העלאה 1 — האתר של app.eiv-music.com
──────────────────────────────────────
Netlify → האתר של app → Deploys → גרור את **כל תיקיית app-site**
(את התיקייה עצמה או את כל הקבצים שבתוכה ביחד)

מה השתנה בנגן הרגיל (index.html):
✓ נכנס אוטומטית מהשער — בלי להקליד קוד פעמיים
✓ הנגן כבר לא משדר אוטומטית לסטרימר! עכשיו יש כפתור
  "📡 שלח לסטרימר" — רק לחיצה עליו מעדכנת את הסטרימר
✓ תמיכה בסטרים ייעודי: אם הגדרת ללקוח Default Stream URL
  בבק אופיס — הסטרימר שלו מנגן אותו אוטומטית
✓ תוקנו פונטים וקבצים שבורים

מה לא השתנה: כל הלינקים הקיימים, מצב סטרימר,
הגבלת מכשירים, תזמונים, הספרייה — הכל כמו היום.

──────────────────────────────────────
העלאה 2 — האתר של ai-rashet.netlify.app
──────────────────────────────────────
Netlify → האתר של נגן ה-AI → Deploys → גרור את **כל תיקיית ai-site**

מה השתנה בנגן ה-AI:
✓ נכנס אוטומטית מהשער (דרך הלינק)
✓ תוקן ה-manifest השבור — עכשיו שמירה למסך הבית
  עובדת כמו שצריך (נוספו manifest.json + אייקונים)
✓ תוקן הפונט השבור

──────────────────────────────────────
העלאה 3 — האתר של admin.eiv-music.com
──────────────────────────────────────
Netlify → האתר של admin → Deploys → גרור את **תיקיית admin**
(אם כבר העלית משלב 1 — אין צורך שוב)

──────────────────────────────────────
בדיקות אחרי ההעלאה (5 דקות)
──────────────────────────────────────
1. שער:  app.eiv-music.com/start.html
   → בחר "נגן רגיל" → קוד מורשה → אמור להיכנס ישר
     לנגן בלי להקליד קוד שוב!
   → בחר "נגן AI" עם קוד שמורשה רק ל-regular → נחסם
   → תן לקוח אחד הרשאת ai בבק אופיס → עכשיו נכנס ל-AI ישר

2. הפרדת סטרימר:
   → פתח נגן בנייד, החלף ערוצים — הסטרימר לא זז!
   → לחץ "📡 שלח לסטרימר" — רק אז הסטרימר מתעדכן

3. סטרים ייעודי:
   → בבק אופיס תן ללקוח Default Stream URL (לינק radio.mp3)
   → פתח את לינק הסטרימר שלו (?id=קוד&mode=streamer)
   → אמור להתחיל לנגן את הסטרים הייעודי אוטומטית

4. לינקים ישנים: פתח לינק סטרימר קיים של לקוח אמיתי
   → ודא שעובד בדיוק כמו קודם

אם משהו לא עובד — Netlify → Deploys → לוחצים על
הפריסה הקודמת → Publish deploy = חזרה אחורה בלחיצה.

──────────────────────────────────────
מה הלאה (שלב 3): ניטור והתראות, גיבוי Supabase
יומי, תיקון קרטועים, והבק אופיס של לינקי הסטרימרים.
